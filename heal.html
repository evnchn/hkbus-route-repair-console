removed CSDI defect patch (OSM-matched)
What this is. CSDI (the government dataset hkbus already uses) stays the baseline for every route. The proposed upstream change is deliberately minimal — one sure-fire defect only: CSDI's routing engine stamped an identical phantom ~10.6 km appendix (落山道 wrong circuit + a loop that wanders to 啟德 and back) onto arrivals at 九龍城碼頭. Exact-vertex stamp matching finds it in 10 route-dirs (6C 6F 2E 5D 11B 61X 75X 85A · A25 22M); each is excised and re-joined with the route's own OSM-matched path (seams ≤ 25 m). Where KMB's line exists (7/10) the healed geometry lands 2–3 m from it, vs 191–739 m before. Two independent mechanisms (stop-span fusion and stamp excision) produce the same healed geometry within 25 m. All other cards below are findings, not applied — kept for evidence and future rounds. This page shows the only places a patch is applied: inter-stop segments where CSDI provably drew a defect — an out-and-back retrace (loop, self-overlap collapse ≥ 20%, the discriminator validated 26/26 vs KMB with 0 false positives) or a grossly inflated wrong circuit (circuit, ≥3× the road alternative and ≥800 m excess). The patch is the OSM map-matched path for that segment only, and is applied only if it passes road/turn QC itself; where the operator's own line (KMB/LWB search API) exists, a patch that disagrees with it is vetoed. Routes with no patch are untouched CSDI, bit for bit; within a patched route only the defective segments change. Routes whose geometry can't be decomposed safely are skipped by design. Suspected wrong-circuits that lack both the self-overlap signature and an operator line to confirm them are held for review (shown amber here) and NOT applied — precision over recall.

Why we're confident the loops are bogus. CSDI geometry is machine-routed, so a mis-solved segment is stamped identically into every route that passes it — a defect recurring across many routes is a data artifact, not a service detour (one cluster repeats in 25 airport E-routes). Clustering the excursions also lets operator-line evidence transfer: a loop refuted by KMB's own line on one route refutes the byte-identical loop on a sibling route that has no oracle. Every card carries its evidence badge.

Maintenance profile: one-off. The patch set is a static artifact; no recurring human effort. Each patch is keyed to the defect it fixes, so if a future CSDI release fixes the defect upstream, the patch retires itself.
